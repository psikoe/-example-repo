Age Quiz Program

User input their ages to get a response from the program.
The age input will be an integer.


Possible outcomes that will be printed.
    Age greater than 100 : "Sorry you're dead."
    Age greater than 40: "You're over the hill."
    Age greater than 65: "Enjoy your retirement!"
    Age less than 13:"You qualify for the kiddie discount." 
    Age is 21: "Congrats on your 21st!"
    Any other age "Age is but a number."
